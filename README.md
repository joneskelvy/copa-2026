<h1 align="center"> Calendário da Copa do Mundo Fifa 2026 </h1>


## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- HTML e CSS
- JavaScript
- Node.js
- Git e Github

## 💻 Projeto

O Calendário da Copa é um projeto que mostra a tabela de jogos da primeira fase da Copa do Mundo Fifa 2026, com um upgrade em relação a versão feita 4 anos antes, dessa vez o objetivo é que se traga os placares em tempo real com atualização do sistema realizando consulta a uma API.
